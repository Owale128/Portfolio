Beskrivning av projektet

Description of the project

Projektet är en webbapplikation som låter användare logga in, registrera sig och söka efter filmer. Användare kan ange sina inloggningsuppgifter för att komma åt systemet eller registrera sig för ett nytt konto om de inte har ett. När användarna väl är inloggade kan de söka efter filmer genom att ange söktermer. Listor över filmer visas baserat på sökningen, och användare kan klicka på en film för att se detaljer. Dessutom kan användare också logga ut från sina sessioner.

The project is a web application that allows users to log in, register, and search for movies. Users can enter their login credentials to access the system or register for a new account if they don't have one. Once logged in, users can search for movies by entering search terms. Lists of movies are displayed based on the search, and users can click on a movie to see details. Additionally, users can also log out of their sessions.

Inloggning och Registrering: Login and Registration:
 Användare kan logga in med sina befintliga konton eller registrera nya. Vid inloggning kontrolleras användarnamn och lösenord. Vid registrering kontrolleras om användarnamnet redan finns och om lösenorden matchar. Users can log in with their existing accounts or new ones. When logging in, the username and password are checked. When registering, it is checked whether the username already exists and whether the password matches.

Sökning efter Filmer: Search for Movies:
 Användare kan söka efter filmer genom att ange en sökterm. Filmlistor visas baserat på sökningen och användare kan klicka på en film för att se detaljer. Users can search for movies by entering a search term. Movie listings are displayed based on the search and users can click on a movie to see details.

Interaktivitet:interactivity:
 Användargränssnittet är interaktivt och responsivt. Det finns feedback för användaraktioner, t.ex. laddningsindikatorer och felmeddelanden. The user interface is interactive and responsive. There is feedback for user actions, e.g. charging indicators and error messages.

  • JavaScript/React
