Beskrivning av projektet

Projektet är en webbapplikation som låter användare logga in, registrera sig och söka efter filmer. Användare kan ange sina inloggningsuppgifter för att komma åt systemet eller registrera sig för ett nytt konto om de inte har ett. När användarna väl är inloggade kan de söka efter filmer genom att ange söktermer. Listor över filmer visas baserat på sökningen, och användare kan klicka på en film för att se detaljer. Dessutom kan användare också logga ut från sina sessioner.

Inloggning och Registrering:

Användare kan logga in med sina befintliga konton eller registrera nya. Vid inloggning kontrolleras användarnamn och lösenord. Vid registrering kontrolleras om användarnamnet redan finns och om lösenorden matchar.

Sökning efter Filmer:

Användare kan söka efter filmer genom att ange en sökterm. Filmlistor visas baserat på sökningen och användare kan klicka på en film för att se detaljer.

Interaktivitet:

Användargränssnittet är interaktivt och responsivt. Det finns feedback för användaraktioner, t.ex. laddningsindikatorer och felmeddelanden.

Teknologier och verktyg

  • HTML
  • MongoDB
  • SCSS/CSS
  • Nodejs/Expressjs
  • JavaScript/React
