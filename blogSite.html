Beskrivning av applikationen

Description of the project

Det här är en bloggplattform utvecklad med React och utrustad med grundläggande CRUD-funktionalitet (skapa, läsa, uppdatera, ta bort). Det tillåter användare att skapa, redigera och ta bort blogginlägg. Användargränssnittet är utformat med formulärfält för inmatning av titel och innehåll, samt knappar för att utföra olika åtgärder på inläggen.

This is a blogging platform developed with React and equipped with basic CRUD (create, read, update, delete) functionality. It allows users to create, edit and delete blog posts. The user interface is designed with form fields for entering title and content, as well as buttons to perform various actions on the posts.

Komponenter:
Components:
 SubmitForm för att lägga till nya inlägg, UpdatePost för att uppdatera befintliga inlägg, DeletePost för att ta bort inlägg och FetchPosts för att hämta befintliga inlägg. SubmitForm to add new posts, UpdatePost to update existing posts, DeletePost to delete posts and FetchPosts to retrieve existing posts.

Ljudeffekter: Sound effects:
 Ljudfiler för att spela upp ljud när olika åtgärder utförs, till exempel när ett inlägg läggs till, uppdateras eller tas bort. Audio files to play sounds when various actions are performed, such as when a post is added, updated or deleted.

Tillståndshantering: Permission management:
 React Hooks för att hantera tillstånd i applikationen som tex blogginlägg, nya inlägg, redigerade inlägg, identifiering av redigerat inlägg, felmeddelanden och referens för det nya inlägget. React Hooks to manage states in the application such as blog posts, new posts, edited posts, identification of edited post, error messages and reference for the new post.

API-anrop: API calls:
 Kommunikation med en backend-API för att utföra CRUD-operationer (skapa, läsa, uppdatera, radera) för blogginlägg. Communication with a backend API to perform CRUD (create, read, update, delete) operations for blog posts.
