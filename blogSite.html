Beskrivning av applikationen

Det här är en bloggplattform utvecklad med React och utrustad med grundläggande CRUD-funktionalitet (skapa, läsa, uppdatera, ta bort). Det tillåter användare att skapa, redigera och ta bort blogginlägg. Användargränssnittet är utformat med formulärfält för inmatning av titel och innehåll, samt knappar för att utföra olika åtgärder på inläggen.

Komponenter:

SubmitForm för att lägga till nya inlägg, UpdatePost för att uppdatera befintliga inlägg, DeletePost för att ta bort inlägg och FetchPosts för att hämta befintliga inlägg.

Ljudeffekter:

Ljudfiler för att spela upp ljud när olika åtgärder utförs, till exempel när ett inlägg läggs till, uppdateras eller tas bort.

Tillståndshantering:

React Hooks för att hantera tillstånd i applikationen som tex blogginlägg, nya inlägg, redigerade inlägg, identifiering av redigerat inlägg, felmeddelanden och referens för det nya inlägget.

API-anrop:

Kommunikation med en backend-API för att utföra CRUD-operationer (skapa, läsa, uppdatera, radera) för blogginlägg.

Teknologier och verktyg

  • React
  • HTML
  • Bootstrap
  • MongoDB
  • SCSS/CSS
  • Nodejs/Expressjs
  • JavaScript/Typescript
