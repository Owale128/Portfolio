Description of the project

Skapade en enkel Todo List webbapplikation som låter användare skapa och hantera en lista över uppgifter att göra. Applikationen tillåter användaren att lägga till nya uppgifter, markera dem som avklarade och ta bort dem från listan. Uppgifterna kan delas in i två kategorier: "Incomplete-List" för uppgifter som ännu inte är slutförda och "Complete" för uppgifter som redan är avklarade.

Created a simple Todo List web application that allows users to create and manage a list of tasks to do. The application allows the user to add new tasks, mark them as completed and remove them from the list. The tasks can be divided into two categories: "Incomplete-List" for tasks that have not yet been completed and "Complete" for tasks that have already been completed.


Lägga till uppgifter:Add details: Användare kan skriva in en ny uppgift i ett textfält och lägga till den i listan genom att klicka på knappen "Add Task". Users can type a new task in a text field and add it to the list by clicking the "Add Task" button.

Markera uppgifter som slutförda:Mark tasks as completed: Genom att klicka på en bockmarkering bredvid varje uppgift kan användaren markera uppgiften som slutförd. By clicking a check mark next to each task, the user can mark the task as complete.

Ta bort uppgifter:Ta bort uppgifter: Användare kan ta bort en uppgift från listan genom att klicka på knappen "Delete" bredvid uppgiften. Users can remove a task from the list by clicking the "Delete" button next to the task.
