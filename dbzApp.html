Beskrivning av projektet

Projektet är en webbapplikation som ger användarna möjlighet att utforska Dragonball Z-karaktärer. Genom att använda en enkel användargränssnitt kan användare se information om karaktärerna och deras bilder.


Hämta Dragonball Z-data:

Applikationen hämtar data om Dragonball Z-karaktärer från Dragonball API och visar dem på sidan.

Visa Dragonball Z-karaktärer:

Karaktärerna presenteras som bilder och namn på sidan.

Interaktivitet:

Genom att klicka på en Dragonball Z-karaktär visas ytterligare information om karaktären i en modalfönster.

Teknologier och verktyg

  • HTML
  • Axios
  • SCSS/CSS
  • JavaScript/Typescript
