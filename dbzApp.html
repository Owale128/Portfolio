Beskrivning av projektet

Description of the project

Projektet är en webbapplikation som ger användarna möjlighet att utforska Dragonball Z-karaktärer. Genom att använda en enkel användargränssnitt kan användare se information om karaktärerna och deras bilder.

The project is a web application that allows users to explore Dragonball Z characters. Using a simple user interface, users can see information about the characters and their images.


Hämta Dragonball Z-data:Download Dragonball Z data:
Applikationen hämtar data om Dragonball Z-karaktärer från Dragonball API och visar dem på sidan.The application retrieves data about Dragonball Z characters from the Dragonball API and displays them on the page.

Visa Dragonball Z-karaktärer:View Dragonball Z Characters:
Karaktärerna presenteras som bilder och namn på sidan. The characters are presented as pictures and names on the page.

Interaktivitet:Interactivity:
Genom att klicka på en Dragonball Z-karaktär visas ytterligare information om karaktären i en modalfönster.Clicking on a Dragonball Z character displays additional information about the character in a modal window.

Teknologier och verktygTechnologies and tools

  • HTML
  • Axios
  • SCSS/CSS
  • JavaScript/Typescript
