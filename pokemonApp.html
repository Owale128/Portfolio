Beskrivning av projektet

Description of the project

Projektet är en enkel webbapplikation som tillåter användare att söka efter Pokémon-kort genom att ange namnet på en Pokémon i en sökruta. När användaren skickar in sökningen hämtas data från Pokémon-tcg.io API:et och resultatet visas på sidan.

The project is a simple web application that allows users to search for Pokemon cards by entering the name of a Pokemon into a search box. When the user submits the search, data is retrieved from the Pokémon-tcg.io API and the result is displayed on the page.

Sökning av Pokémon: Finding Pokemon:
 Användaren kan skriva in namnet på en Pokémon i sökrutan och skicka in för att få relaterade Pokémon-kort. The user can enter the name of a Pokemon in the search box and submit to get related Pokemon cards.

Visa resultat: Show result:
 Sökresultatet: visas som bilder av Pokémon-kort på sidan. Search results are displayed as images of Pokemon cards on the page.

Interaktivitet: interactivity:
 Klicka på ett Pokémon-kort för att aktivera en visuell effekt på bilden. Click on a Pokemon card to activate a visual effect on the image.

Teknologier och verktyg Technologies and tools

  • HTML
  • Axios
  • SCSS/CSS
  • JavaScript/Typescript
