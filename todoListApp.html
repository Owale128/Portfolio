Beskrivning av projektet

Description of the project

Det här projektet är en enkel att göra-lista skriven i React. Användaren kan lägga till nya uppgifter, markera dem som slutförda och ta bort dem från listan. Projektet använder sig av en backend-tjänst för att spara och hämta uppgifterna.

This project is a simple to-do list written in React. The user can add new tasks, mark them as completed and remove them from the list. The project uses a backend service to save and retrieve the data.

Hämta uppgifter: Fetch information:
 Uppgifterna hämtas från en backend-tjänst när komponenten renderas för första gången. The data is retrieved from a backend service when the component is rendered for the first time.

Lägga till uppgift: Add task:
 Användaren kan skriva in en ny uppgift i ett formulär och lägga till den i listan genom att trycka på knappen "Add Todo". The user can enter a new task in a form and add it to the list by pressing the "Add Todo" button.

Markera uppgift som slutförd: Mark task as completed:
 Användaren kan markera en uppgift som slutförd genom att klicka på en kryssruta bredvid uppgiftstiteln. Detta ändrar statusen för uppgiften i backend-tjänsten. The user interface is interactive and responsive. There is feedback for user actions, e.g. charging indicators and error messages.

Ta bort uppgift: Remove task:
 Användaren kan ta bort en uppgift från listan genom att klicka på en "Delete"-knapp bredvid uppgiftstiteln. The user can remove a task from the list by clicking a "Delete" button next to the task title.
