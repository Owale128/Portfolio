Beskrivning av projektet

Description of the project

Detta är en webbapplikation som låter användare spela en fråge-quiz. Den håller koll på frågor, användarens svar, poäng och visar resultat. Dessutom kan användare ange sitt namn innan de börjar spelet. Koden innehåller även funktioner för att starta om eller avsluta quizen och visar en lista över högsta poäng. Det finns också en modal som visar rätt svar efter att användaren har svarat på frågorna.

This is a web application that allows users to play a question quiz. It keeps track of questions, user answers, scores and displays results. Additionally, users can enter their name before starting the game. The code also includes functions to restart or exit the quiz and display a list of the highest scores. There is also a modal that displays the correct answer after the user answers the questions.

Funktionalitet:Functionality:
 Koden innehåller logik för att hantera frågor och svar, inklusive poängräkning. Dessutom kontrolleras användarnamnet innan quizen startar. Användaren har möjlighet att starta om quizet och avsluta det när som helst. Frågor hämtas från en källa och blandas för varje ny omgång. Resultatet av quizen skickas till ett API för ytterligare hantering. Dessutom finns det funktioner för att kontrollera och visa den högsta poängen samt för att visa modaler som visar de korrekta svaren. The code contains logic for handling questions and answers, including scoring. In addition, the username is checked before the quiz starts. The user has the option to restart the quiz and end it at any time. Questions are drawn from a source and shuffled for each new round. The result of the quiz is sent to an API for further processing. Additionally, there are functions to check and display the highest score as well as to display modals showing the correct answers.
